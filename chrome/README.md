## Shortcuts

### view page source

    command-control u

### open dev tools

    command-control i

### open JS console

    command-control j

### inspection mode

    command-shift c

### Undo Close Tab

    ctl-shift T

OS X

    command-shift T

### tab navigation
    
    control-tab    // next tab
